# Senseimath 🎓

Платформа для репетиторов и учеников: расписание, интерактивная доска, файлообмен и уведомления.

## 🚀 Быстрый старт

### Требования
- Node.js 18+ или Bun
- npm, yarn или bun

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/DijiQ/Senseimath_z.git
cd Senseimath_z

# Установить зависимости
npm install
# или
bun install

# Создать .env файл
cp .env.example .env

# Инициализировать базу данных
npm run db:push
# или
bun run db:push

# Запустить в режиме разработки
npm run dev
# или
bun run dev
```

Откройте **http://localhost:3000** в браузере.

## 📋 Функции

### Для репетитора
- 📅 **Календарь** — планирование уроков по дням и неделям
- 👥 **Управление учениками** — поиск, приглашение, список
- 📝 **Создание уроков** — дата, время, длительность, ученики
- 📋 **Дублирование расписания** — копирование на следующую неделю
- 🎨 **Интерактивная доска** — рисование, фигуры, сохранение
- 📁 **Файлообмен** — загрузка материалов к урокам
- 🔔 **Уведомления** — о приглашениях, уроках, файлах

### Для ученика
- 📅 **Расписание** — просмотр своих уроков
- 👨‍🏫 **Репетиторы** — список и приглашения
- ✅ **Приглашения** — принятие или отклонение
- 🎨 **Доска** — просмотр сохранённых досок
- 📁 **Файлы** — скачивание материалов
- 🔔 **Уведомления** — о новых уроках и файлах

## 🛠 Технологии

- **Next.js 16** — React фреймворк
- **TypeScript** — типизация
- **Prisma** — ORM для базы данных
- **SQLite** — база данных
- **Tailwind CSS** — стили
- **shadcn/ui** — UI компоненты
- **Zustand** — состояние
- **date-fns** — работа с датами
